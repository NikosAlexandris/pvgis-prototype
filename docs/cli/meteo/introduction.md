Meteorological variables make part of the algorithmic backbone in PVGIS.
Ambient temperature and wind speed impact the photovoltaic performance.
A series of meteorogical time series are assessed statistically to assemble the
so called Typical Meteorological Year.

``` bash
pvgis-prototype meteo introduction
```
