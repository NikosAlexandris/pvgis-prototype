Install the required packages via 

  <div class="termy">

  ``` console
  pip install .[web]
  ```
  </div>
  
This will install among other packages [FastAPI][FastAPI] and [`uvicorn`][uvicorn].

[FastAPI]: https://fastapi.tiangolo.com/
[uvicorn]: https://www.uvicorn.org/
